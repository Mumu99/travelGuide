# 南京

# 第一天

路线

机场✈️：南京禄口机场 乘坐 s1 号线(南京南方向)7 站到达 南京南站 内换乘 1 号线(迈皋桥方向)7 站到达 张府园
高铁🚅： 南京站地铁3号线 往 林场方向 乘坐 6 站到 夫子庙
居住🏪：夫子庙 附近，离游玩距离适中。当天晚上可以去大成殿(纪念孔夫子地方,有灯光)

# 第二天

行程安排

古鸡鸣寺 → 明城墙 → 玄武湖 → 总统府 → 新街口吃吃吃

路线

  1. 步行到 1 号线张府园 (地铁站，迈皋桥方向)3 站到达鼓楼 换乘 4 号线(仙林湖方向)1 站到鸡鸣寺(5 号口) 下车之后步行 500 米到达。
  2. 在鸡鸣寺游玩之后可以直接步行到明城墙 。
  3. 步行到鸡鸣寺(地铁站)乘坐 3 号线(林场方向)1 站到南京林业大学新庄下车(1 号口)步行 1km 到达玄武湖。
  4. 步行到南京林业大学新庄乘坐地铁 3 号线(秣周东路)3 站到大行宫(5 号口)之后步行 400 米到达总统府 。
  5. 从总统府 步行(1.3km)到达新街口 。
  6. 步行回去

# 第三天

行程安排

新街口继续吃 → 南京大屠 杀遇难同胞纪念馆(需提前预约)→ 夫子庙 → 秦淮河 → 老门东

路线

  1. 可步行到新街口 / 步行至张府园 乘坐 1 号线(迈皋桥方向)1 站到新街口(2/4 号口出)步行 400 米到终点，吃完早饭。
  2. 步行到新街口地铁口，乘坐 2 号线(油坊桥方向)4 站到达云锦路(2 号口) 之后步行 100 米到达终点。
  3. 步行到南京大屠杀遇难同胞纪念馆乘坐环城游 6 号线 3 站到达夫子庙瞻园，之后步行 500 米到达。
  4. 步行 400 米到达秦淮河(以及附近的江南贡院)。
  5. 步行 1.2km 到老门东 (吃饭，喷泉)(陆氏梅花糕\黄勤记的鸡汁豆腐\黄桥烧饼和牛肉锅贴)。
  6. 之后打车回去(张府园 )

# 第四天

行程安排

德基广场 → 先锋书店 → 南京博物馆 →1912 酒吧街区

路线

  1. 步行至张府园 乘坐 1 号线(迈皋桥方向)1 站到达新街口(7 号口)之后步行 200 米到达，吃完早饭。
  2. 步行到新街口地铁口，乘坐 1 号线(迈皋桥方向)1 站到达珠江路之后步行 1.4km 到达终点(可以买周边)。
  3. 步行 1.2km 到中山陵停车场乘坐 34 路(中山码头方向)5 站到中山门，下车步行 50 米到达。
  4. 步行 500 米到明故宫2 号线(1 号口进去,油坊桥方向)2 站到大行宫(5 号口出)步行 500 米达到。
  5. 之后打车回去(张府园 )

# 第五天

行程安排

睡懒觉 → 红山森林动物园

路线

步行至张府园 乘坐 1 号线(迈皋桥方向)7 站到红山动物园(2 号口)步行 600 米到达。
