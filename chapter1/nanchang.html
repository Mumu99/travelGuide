# 南昌

# 第一天

行程安排

家 → 南昌 → 万寿宫（珠宝街） → 秋水广场 → 八一大桥

路线

  1. 自驾、快的 行程大约3h
  2. 居住🏪：凯瑞酒店 新万达广场附近（位置较居中，附近商城较多，出行方便，具体酒店视情况而定）需提前10天或者1周左右预定。
  3. 自驾到万寿宫（珠宝街）可先游玩之后再到小吃街吃东西
    1. 南昌水煮
    2. 皮蛋瓦罐汤
    3. 拉面麻油小包子
    4. 老太婆辣藕
    5. 南昌拌粉
    6. 串串
    7. 麻糍
    8. 糯叽叽的热罩子
    9. 腊肠蛋黄糯米饭
    10. 糊羹
    11. 芋泥麻糍
    12. 竹筒奶茶（鲲茶）
    13. 绿豆饼
    14. 卷满肉泥的花卷
  4. 晚上可以去秋水广场看喷泉 (第一场：19:30 - 19:45、第二场：20:30 - 20:45) 具体时间得再查
    1. 景点位置：江西省南昌市东湖区雄州路与赣江中大道交叉口
  5. 散步八一大桥有点长，视情况而定

# 第二天

行程安排

江西省博物馆 → 羊子巷 → 滕王阁游轮 → 新万达

路线

  1. 自驾到江西省博物馆。
    1. 周一闭馆（法定节假日除外）
    2. 开放时间：9:00 - 17:00
    3. 需要携带身份证
    4. 小程序：江博预约进行预约
    5. 景点位置：1号线珠江路口站
  2. 自驾到羊子街
    1. 小吴水煮
    2. 芋泥麻糍（有相同）
    3. 麻辣肠子、麻辣莲藕
    4. 鸡蛋汉堡
    5. 汤泡饭
    6. 粉蒸肉
    7. 咸蛋黄糯米饭（有相同）
    8. 咸菜笋馅的包子
    9. 皮薄馅大的烧饼
    10. 甜豆花
    11. 野生小板栗
    12. 韭菜大饼
  3. 自驾到滕王阁夜间游轮。
    1. 景点位置：江西省南昌市东湖区仿古街58号
    2. 取票地点：东湖区榕门路仿古街76号，滕王阁游轮售票处
    3. 时间：晚间第一场：19:00 - 20:45
    4. 购票：南昌本地宝：滕王阁游轮进行购票 - 二楼贵宾厅第一班
  4. 自驾到新万达可以购物和看电影。

# 第三天

行程安排

第一种：八一起义纪念馆 → 绳金塔 → 699文化创意园 → 象湖公园（晚上） → 国贸（晚上）
第二种：八一起义纪念馆 → 动物园
第三种：八一起义纪念馆 → 国贸

路线

  1. 自驾到八一起义纪念馆。
    1. 周一闭馆（法定节假日除外）
    2. 开放时间：9:00 - 17:00
    3. 需要携带身份证
    4. 小程序：byg人脸识别进行预约
    5. 景点位置：地铁1号线八一馆站
  2. 自驾到绳金塔之后可以在绳金塔美食街吃东西。
  3. 自驾40min左右动物园（开放时间：8:30 - 18:30、需要携带身份证、小程序：南昌动物园进行买票）
  4. 自驾到699文化创意园适合拍照。
  5. 自驾到象湖公园（晚上）
    1. 景点位置：南昌市青云谱区施尧路
  6. 自驾到国贸（晚上）

# 注意事项

相关注意事项

  1. 携带物品
    1. 衣服（根据当时气温和天气而定）
    2. 身份证
    3. 牙膏牙刷、毛巾、沐浴露、洗发水(可当天临时购买)
    4. 充电宝、雨伞、现金（以备不时之需）、三脚架
  2. 交通相关
    1. 微信：腾讯出行 - 公交地铁
    2. 支付宝：洪城一卡通 - 公交地铁
  3. 饮食相关
    1. 南昌老三样
    2. 拌粉、瓦罐汤
    3. 赣菜
